HTML / CSS / JS Rediscovered!

A log of my late 2022 journey through ground-up (but non-blocking and persistent) studies of web foundations in my pursuit of becoming a front-end developer and attaining an understanding of the MERN stack.

Initiated by my ongoing runthrough of the MDN Front-end Web Developer Guide.

Its core topics are:

  1. HTML - Structuring the web
  2. CSS - Styling the web
  3. JavaScript - Dynamic client-side scripting
  4. Web forms - Working with user data
  5. Accessibility - Make the web usable by everyone
  6. MathML - Writing mathematics with MathML
  7. Tools and testing
  8. Server-side website programming
  9. Further resources

... but my focus areas will be on CSS, JavaScript, forms, tools (specifically frameworks) and server-side, essentially refocusing my pursuit of MERN fluency.
